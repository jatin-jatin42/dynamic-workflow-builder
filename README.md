# VectorShift Node Canvas

A dynamic, interactive workflow builder featuring a React Flow-based canvas with robust Node abstractions, a modern SaaS UI, and a FastAPI backend for Directed Acyclic Graph (DAG) analysis.

## How to start the backend

```bash
cd backend
source venv/bin/activate
uvicorn main:app --reload
```

## How to start the frontend

```bash
cd frontend
npm install
npm start
```
